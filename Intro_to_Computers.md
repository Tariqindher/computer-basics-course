# 📦 Module 1: Understanding Computer, Internet & Developer Tools

## 🖥️ Basic Parts of a Computer

### 🧩 Motherboard
- The main board of the computer.
- Connects all computer parts together.

### 🧠 CPU (Central Processing Unit)
- The brain of the computer.
- Executes instructions and performs calculations.

### ⚡ RAM (Random Access Memory)
- Temporary memory.
- Stores data while programs are running.
- Data is deleted when the computer is turned off.

### 💾 HDD (Hard Disk Drive)
- Permanent storage.
- Stores files even after shutdown.

### 🚀 SSD (Solid State Drive)
- Faster than HDD.
- Stores data permanently.

### 🔌 PSU (Power Supply Unit)
- Supplies power to all computer components.

### 🎮 GPU (Graphics Processing Unit)
- Creates graphics, videos, and games.

---

# 📁 Module 2: Working with Files, File Systems & Media Formats

## 📂 File Naming Best Practices

Good file names make projects easy to understand.

✅ Use descriptive names

Example:

```
about-us.html
contact.html
login-page.html
```

❌ Avoid names like:

```
page1.html
test123.html
newfile.html
```

---

## 📁 Root Directory

The **Root Directory** is the main folder of a project.

All project files and folders are inside it.

Example:

```
MyProject/
│
├── index.html
├── style.css
├── script.js
└── images/
```

---

## 🔍 Smart Search Tips

### Search inside one website

```
site:freecodecamp.org javascript
```

---

### Search specific file types

```
filetype:pdf javascript
```

---

### Exclude words

```
python -django
```

Shows Python results without Django.

---

### Include words

```
react +hooks
```

Finds results that include Hooks.
