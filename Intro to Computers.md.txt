### ??? What are the Basic Parts of a Computer?

- ?? Motherboard ? Main board that connects all parts  
- ?? CPU ? Brain of the computer that processes instructions  
- ? RAM ? Temporary memory for fast work  
- ?? Hard Drive (HDD/SSD) ? Stores data permanently  
- ?? PSU ? Gives power to the computer  
- ?? GPU ? Handles graphics and visuals  
- ?? Network Card ? Connects computer to internet  

---

### ??? Keyboard, Mouse & Devices Usage

- ??? Hold mouse gently to avoid hand pain  
- ?? Use keyboard shortcuts to save time  
- ?? Maintain correct posture to avoid health issues  
- ?? Take regular breaks to reduce stress on body  
- ?? Use ergonomic tools for comfort  

---

### ?? Types of Internet Service Providers (ISP)

- ?? Tier 1 ISP ? Big global internet companies  
- ?? Tier 2 ISP ? Country-level providers  
- ?? Tier 3 ISP ? Local internet companies  

#### ?? Internet Types:

- ?? Fiber ? Fastest internet using light signals  
- ?? Cable ? Uses TV network cables  
- ?? DSL ? Uses telephone lines  
- ?? 5G Internet ? Uses mobile towers  
- ??? Satellite ? Internet from space satellites  

---

### ?? Safe Ways to Sign Into Computer

- ?? Always use strong passwords  
- ? Avoid simple passwords like 12345  
- ??? Use 2-factor authentication (2FA)  
- ?? Use fingerprint or face ID if available  
- ?? Keep your system locked when not in use  

---

### ?? Developer Tools Used by Professionals

- ?? Computer ? Main development machine  
- ????? Code Editors ? VS Code, IntelliJ, PyCharm  
- ?? Git ? Tracks code changes  
- ?? GitHub ? Stores and shares code online  
- ?? Package Managers ? Manage libraries (npm, pip, etc.)  
- ?? Testing Tools ? Check if code works properly  
- ?? Browsers ? Chrome, Firefox for testing websites
